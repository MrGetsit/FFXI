# Cooldown Tracker
<img width="182" height="83" alt="CooldownTracker" src="https://github.com/user-attachments/assets/5cbfba8a-a053-44e4-916a-4d199a246b75" />

Shows spell and ability recast timers for all multibox characters. Able to filter abilities you don't want to see the timers for.

### Command Usage:
```
  cd pos <x> <y>       -- Set display position
  cd size <pixels>     -- Set font size

  cd add <ability>     -- Add ability to filtered list (Don't show timer for ability)
  cd remove <ability>  -- Remove ability from filtered list
  cd list              -- Show all filtered abilities
  cd clear             -- Clear filtered list
```
